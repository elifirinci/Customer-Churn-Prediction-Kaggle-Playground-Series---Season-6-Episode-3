# 📊 Customer-Churn-Prediction-Kaggle-Playground-Series---Season-6-Episode-3

This project focuses on predicting customer churn using machine learning techniques. The goal is to identify customers who are likely to leave a service, enabling businesses to take proactive retention measures.

---

## 🚀 Project Overview

Customer churn prediction is a critical task for businesses aiming to improve customer retention and maximize revenue. In this project, we:

* Analyze customer behavior data
* Perform data preprocessing and feature engineering
* Train and evaluate machine learning models
* Generate predictions on unseen data

---

## 📂 Dataset

The dataset used in this project is provided by Kaggle:

👉 [Playground Series S6E3 Dataset](https://www.kaggle.com/competitions/playground-series-s6e3/data?utm_source=chatgpt.com)

It contains structured customer-related features such as demographics, usage patterns, and subscription details.

---

## 🧠 Models Used

The notebook explores various machine learning algorithms, including:

* Logistic Regression
* Decision Trees
* Random Forest
* Gradient Boosting (e.g., XGBoost / LightGBM)

Model performance is evaluated using appropriate metrics such as:

* Accuracy
* Precision / Recall
* F1 Score
* ROC-AUC
